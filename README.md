# kele-code

An agentic developer CLI tool for software engineering tasks.

## Features

- File reading, editing, and creation
- Code search (ripgrep-based grep + glob)
- Shell command execution
- Web fetch and search
- Sub-agent spawning and multi-agent coordination
- Skill system for reusable workflows
- MCP (Model Context Protocol) server integration
- IDE bridge (VS Code, JetBrains)
- Vim mode, voice input, keybinding customization

## Tech Stack

| Category | Technology |
|---|---|
| Runtime | Bun |
| Language | TypeScript |
| Terminal UI | React + Ink |
| CLI Parsing | Commander.js |
| Schema Validation | Zod v4 |
| Code Search | ripgrep |
| Protocols | MCP SDK, LSP |
